Endpoint

POST /api/extract

Extract structured data from a chart image. Supports bar charts, line charts, and pie charts.

Request Parameters

Parameter Type Required Description
file File Yes Chart image file (PNG, JPG, JPEG)
chart_type String No Chart type: "Auto", "Bar", "Line", or "Pie" (default: "Auto")
min Number No Minimum Y-axis value for rescaling
max Number No Maximum Y-axis value for rescaling
debug Boolean No Enable debug mode (saves intermediate images). Use "true" or "false" (default: "false")

Example Request (cURL)

curl -X POST http://localhost:8000/api/extract \ -F "file=@chart.png" \ -F "chart_type=Auto" \ -F "debug=false"

Example Request (Python)

import requests url = "http://localhost:8000/api/extract" files = {"file": open("chart.png", "rb")} data = { "chart_type": "Auto", "debug": "false" } response = requests.post(url, files=files, data=data) result = response.json() print(f"Chart Type: {result['chart_type']}") print(f"Data Points: {len(result['data'])}")

Example Request (JavaScript)

const formData = new FormData(); formData.append('file', fileInput.files[0]); formData.append('chart_type', 'Auto'); formData.append('debug', 'false'); fetch('http://localhost:8000/api/extract', { method: 'POST', body: formData }) .then(response => response.json()) .then(data => { console.log('Chart Type:', data.chart_type); console.log('Data:', data.data); });

Response Format

Success Response (Bar Chart)

{ "status": "success", "chart_type": "Bar", "data": [ { "category": "2020", "label": "Revenue", "value": 150000, "color": "#FF0000" }, { "category": "2021", "label": "Revenue", "value": 180000, "color": "#FF0000" } ], "metadata": { "titles": { "chart": "Annual Revenue", "value_axis": "Amount ($)", "category_axis": "Year" }, "y_axis_range": { "min": 0, "max": 200000, "source": "detected" } }, "csv_path": "output_fixed/result.csv" }

Success Response (Line Chart)

{ "status": "success", "chart_type": "Line", "data": [ { "category": "Jan", "label": "Sales", "value": 45.5, "x_pixel": 100, "y_pixel": 200, "color": "#0000FF" } ], "metadata": { ... } }

Success Response (Pie Chart)

{ "status": "success", "chart_type": "Pie", "data": [ { "category": "Product A", "percentage": 35.5, "angle": 127.8 }, { "category": "Product B", "percentage": 28.3, "angle": 101.88 } ], "metadata": { ... } }

Error Response

{ "status": "error", "error": "No file uploaded", "message": "Please provide a chart image file" }

Response Fields

Field Type Description
status String "success" or "error"
chart_type String Detected or specified chart type
data Array Extracted data points (structure varies by chart type)
metadata Object Additional information (titles, axis ranges, etc.)
csv_path String Path to saved CSV file (if applicable)
